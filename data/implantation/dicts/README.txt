README
======

Este documento intentará la necesidad, uso, y beneficios de los diccionarios
asociativos que en este directorio se encuentran.

aux-departamento-materia
------------------------

    Asociación entre una materia y un departamento.
    Se utiliza en la generación del SQL de materias.

aux_materia-codigo
------------------

    Asociacion de una materia que engloba sus diferentes codigo, que pueden ser
    diferentes entre planes de estudio.
